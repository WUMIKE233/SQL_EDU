# 前言

SQL 是数据库课程中最重要的实践语言之一。对本科生而言，学 SQL 不是为了背诵语法，而是为了形成一种“用数据描述现实、用查询回答问题”的能力。

这份教材以教学可用为目标，遵循以下原则：

- 先讲问题，再讲语法，避免只记命令不会建模
- 尽量使用小而完整的校园案例，降低理解门槛
- 每章都给出可运行示例，鼓励边学边做
- 将“设计、查询、更新、事务、安全”串成一条完整学习链路

## 你将学到什么

完成本教材后，你应当能够：

- 理解关系模型、表、元组、属性、主键、外键等基本概念
- 使用 SQL 完成建库建表、插入更新、单表查询、多表连接和统计分析
- 理解事务、并发、约束、索引和视图的基本作用
- 设计一个小型业务数据库，并为其编写核心查询

## 建议学习方式

1. 先通读每章的概念部分，理解语法解决的实际问题。
2. 再亲手运行示例 SQL，观察结果。
3. 最后独立完成习题，尝试不用“照抄”完成需求。

## 配套案例

教材统一使用“高校教学管理系统”作为主案例，包含以下实体：

- `departments`：院系
- `teachers`：教师
- `students`：学生
- `courses`：课程
- `enrollments`：选课记录

这个案例足够简单，适合入门；同时又包含多表关系、约束、统计与事务，足以覆盖本科 SQL 课程的大部分核心内容。
