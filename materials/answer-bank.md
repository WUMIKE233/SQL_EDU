# SQL EDU 答题库说明

本答题库基于以下材料整理：

- `数据库PTA习题.pdf`：主要补充存储过程与触发器、规范化、事务与恢复、SQL 编程题。
- `复习课件1.pptx`：主要补充数据库系统概述、关系模型、关系代数、SQL 语言、视图。
- `复习课件2.pptx`：主要补充数据库安全、游标、存储过程、事务恢复、并发控制、函数依赖与范式。
- 微信截图 `第3周`：补充数据类型、主键/候选码、三级模式、DDL、索引、建表约束题。
- 微信截图 `第6周`：补充子查询、连接、BETWEEN、LIKE、UPDATE/DELETE、聚合、SQL 查询题。
- 新增复习题截图：补充完整性、排序、级联删除、封锁协议、死锁、游标、视图与综合 SQL 编程题。
- 仓库原有 `examples/practice.sql`：用于对 SQL 查询题的风格保持一致。

## 当前覆盖

- 题目总数：121 题。
- 题型：判断题、单选题、填空题、SQL 题、简答题。
- 专题：SQL语言、关系模型、规范化、事务恢复、存储过程与触发器、视图与权限。

## 使用方式

题库主数据位于 `src/data/questionBank.ts`。每道题包含：

- `stem`：题干。
- `correctAnswer` / `modelAnswer`：标准答案与展示答案。
- `explanation`：解析。
- `source`：来源文件与页/幻灯片定位。
- `tags`：知识点标签。

PDF 和微信截图中的部分题目没有可提取文本层，已通过图片页人工核读整理；对长填空题中可接受答案较多的题目，前端以参考答案和解析形式呈现。
